# 🪑 Меблерія — Магазин твого затишку

**Меблерія** — це сучасний вебдодаток для перегляду та замовлення меблів. Наша місія — поєднувати сучасний дизайн, довговічність та доступність, допомагаючи кожному створити ідеальний інтер'єр.

## 👥 Про команду
* **Назва команди:** ROZBYSHAKY
* **Репозиторій:** project-group-12
* **Склад команди:** 11 учасників.

## 🛠 Технологічний стек
Проєкт виконано з використанням сучасних стандартів веб-розробки:
* **Markup:** HTML5 (робота з SVG-спрайтом).
* **Styles:** CSS.
* **Scripting:** JavaScript.
* **Libraries:**
    * <u>Swiper.js</u> — для реалізації слайдерів у блоках відгуків та популярних товарів.
    * <u>iziToast</u> — для обробки сповіщень при роботі з формами.
    * <u>Accordion</u> — для секції FAQ.
    * <u>CSS Star Rating</u> — для відображення рейтингу товарів.

## 🏗 Структура проєкту
Згідно з архітектурою файлів:
* `src/css/` — стилі для кожного блоку окремо (hero.css, furniture-list.css тощо). 
    * `базові налаштування` — base.css, reset.css, container.css.
* `js/` — модульна логіка:
    * `api/` — взаємодія з бекендом.
    * `render/` — функції для динамічного створення карток та списків.
    * `utils/` — допоміжні скрипти.
* `partials/` — HTML-фрагменти для кожного блоку окремо (hero.html, furniture-list.html тощо)..
* `img/` — оптимізовані зображення, адаптовані під Retina-дисплеї.

## 🚀 Функціональні можливості
1.  **Адаптивність:** Сайт повністю коректно відображається на Mobile (від 375px), Tablet (768px) та Desktop (1440px).
2.  **Каталог меблів:** Динамічне завантаження товарів з API, фільтрація за категоріями та пагінація "Показати ще".
3.  **Детальна інформація:** Модальне вікно з галереєю, описом, характеристиками та вибором кольору.
4.  **Зворотний зв'язок:** Форма замовлення з валідацією та відправкою даних на сервер.
5.  **Інтерактивність:** Плавний скрол по якірних посиланнях, інтерактивне бургер-меню, акордеони в FAQ та слайдер відгуків.

## 🔌 Інтеграція з Backend
Проєкт використовує зовнішній API для отримання даних:
* **Base URL:** `https://furniture-store-v2.b.goit.study/api-docs/`
* **Ендпоінти:**
    * <u>/furniture</u> — список усіх товарів.
    * <u>/categories</u> — перелік доступних категорій для фільтра.
    * <u>/feedbacks</u> — відгуки клієнтів.
    * <u>/orders</u> — створення нових замовлень.

## ⚙️ Як запустити проєкт локально

1.  **Клонуйте репозиторій:**
    ```bash
    git clone https://github.com/[ваш-аккаунт]/project-group-12.git

    Використовуйте SSH key.
    ```
2.  **Перейдіть у папку проєкту:**
    ```bash
    cd project-group-12
    ```
3.  **Встановіть залежності (якщо використовується збірник, наприклад Vite або Parcel):**
    ```bash
    npm install
    ```
4.  **Запустіть режим розробки:**
    ```bash
    npm run dev
    ```
5.  **Відкрийте в браузері:** `http://localhost:5173` (або інший порт, вказаний у консолі).
